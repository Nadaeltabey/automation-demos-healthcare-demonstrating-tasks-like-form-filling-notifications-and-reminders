# Automation Demos - healthcare.gammal.tech

This repository contains simple automation scripts demonstrating tasks such as:
- Filling forms automatically
- Sending notifications
- Scheduling daily reminders

These scripts are part of my training at healthcare.gammal.tech to showcase skills in automating repetitive tasks and improving efficiency.

## How to run
1. Navigate to the folder of the script.
2. Run with Python 3:
   ```bash
   python form_filling/fill_form.py
   python notifications/send_notification.py
   python reminders/daily_reminder.py
   ```
